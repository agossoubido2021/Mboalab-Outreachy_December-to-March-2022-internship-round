 Installation :
 Inscrire vos paramétres de connection dans le fichier admin/connection.php
 Insérer le fichier mysql.sql dans votre base de donnée
 Par défault, pour accéder à la partie administration, vous avec comme pseudo : admin et comme mot de passe : admin
 Pour modifier le titre de votre site, aller dans admin et modifier le contenue de titre.txt
